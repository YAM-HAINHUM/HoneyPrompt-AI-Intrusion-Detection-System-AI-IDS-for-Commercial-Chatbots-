# HoneyPrompt-AI-Intrusion-Detection-System-for-Commercial-Chatbots-
🐝 HoneyPrompt is an AI Intrusion Detection System (AI-IDS) that protects commercial chatbot applications from prompt injection, jailbreak, and system prompt extraction attacks using cybersecurity-inspired honeypot techniques, intelligent risk scoring, deception-based responses, and real-time attack telemetry.
